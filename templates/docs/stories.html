    As a visitor, I want to register, so I can log into my account.

      Acceptance Criteria

    1. Inputs for 5 fields: username, handle, email, password, and password confirmation
    2. Password must be > 7 characters, contain one uppercase charater, and one special character
    3. Failure to meet password criteria alerts the user to the error
    4. Email must be unique
    5. Handle must be unique
    6. All fields are required
    7. Successful registration redirects to an account page where more information is requested
